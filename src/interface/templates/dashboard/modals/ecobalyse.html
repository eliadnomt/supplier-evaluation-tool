How This Tool Uses Ecobalyse

This tool integrates with the Ecobalyse API to estimate environmental impact scores for textile suppliers. The Ecobalyse score provides a comprehensive environmental assessment based on multiple factors including material composition, production processes, and geographic locations. For more information, visit the official Ecobalyse simulator.

Not all material types are available in the Ecobalyse simulator at this stage (and therefore this tool) due to limited or lacking data.

Assumed Parameters

In Ecobalyse, durability is one of the parameters used to evaluate the overall impact of a garment. Because Bourrienne is classified as a small business, its “repair incentive” score—part of the non-physical durability coefficient—is calculated solely from the ratio between the average repair cost and the garment’s reference sale price.

When dyeing details are missing for a supplier, the tool simply defaults to Ecobalyse’s “average dyeing” profile, which combines the impacts of discontinuous dyeing (more water and energy, low dye concentration) and continuous dyeing (less water but higher, more polluting dye concentrations). This gives a realistic baseline when the exact process is unknown.
